# ## üî∑ Client Support

### Windows 11 23H2 Support  
- Full client qualification for **Windows 11 23H2**.  
- Windows 11 builds **22H2** and **21H2** continue to run, but are **not recommended** because they no longer receive security updates from Microsoft.  


---

### macOS 15 (Sequoia) Support  
- Adds support for **macOS 15 Sequoia** across:
  - Avid NEXIS Client Manager  
  - File Gateway  
  - Editorial workflows (MC, PT)  


---

### Ubuntu 22.04 & Rocky Linux Client Support  
- Expanded support for Linux clients used in hybrid and cloud workflows.  
- Applies to:
  - File Gateway  
  - Cloud UX environments  
  - MediaCentral workflows

# ## üî∑ Storage & Capacity

---

# ## üî∑ Cloud Storage

- Cloud systems updated to support **600 active clients**.  
- Matches updates originally introduced in 2024.11 and forward-ported to the 2025.x line.

# ## üî∑ Summary

Avid NEXIS **2025.5.0** introduced:

- Windows 11 23H2 support  
- macOS 15 Sequoia support  
- Linux client additions

Use **2025.5.1 or later**.

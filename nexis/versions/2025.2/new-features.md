# New for Nexis version 2025.2.0 – withdrawn

# WARNING! - Withdrawn due to an issue that may prohibit write operations. Avid recommends upgrading to Avid NEXIS version 2025.2.2 as soon as possible. All features and fixes introduced in 2025.2.0 are included in 2025.2.2. Avid NEXIS version 2025.2.0 is a major release that includes the following: l Support for Shared Delete l Larger drive capacities for Avid NEXIS F2/F2X, F5, and F5 | NL l Support for Rocky Linux clients with 40GbE connection l Removal of the Avid NEXIS Client installer for CentOS from the cloud l Support for custom icons on macOS folders


# New for Nexis version 2025.2.2
Includes all features and fixes from 2025.2.0, along with additional bug fixes.  Support for macOS 15 Sequoia is not included in the client installer for this release. For macOS Sequoia clients, use Avid NEXIS Client v2025.3.0 or higher.

# New Features for Nexis versions 2022.5.0 to 2025.5.10 – latest version (as of 30th Oct 2025)

Note: An Avid NEXIS server and client compatibility issue has been identified that can cause media consistency errors when using Avid NEXIS 2023.8 or a newer client with Avid NEXIS 23.8 or an older server. If the issue occurs, it will most likely impact workflows where small incremental changes are made frequently. Although infrequent, it may cause portions of media to be unreadable.

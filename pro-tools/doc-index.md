# Pro Tools — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base Pro Tools documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Pro-Tools-Documentation>

Page last updated: 16/12/2025.

> **Structural note — OS-split ReadMes:** Unlike Media Composer, Pro Tools ships separate ReadMe
> PDFs for macOS and Windows. Both are listed for each version where available.

> **Structural note — ZIP bundles:** Each major release includes a "Release Documents - ALL"
> ZIP archive containing all PDFs for that version. Listed per release below.

> **Structural note — Release Notes as KB articles:** Release Notes for some versions link to
> KB articles (HTML, not PDF) rather than to standalone PDFs. These are listed separately.

> **Structural note — Shortcuts Guide:** A dedicated Shortcuts Guide PDF is published per major
> release. Listed from 2024.6 onwards; absent from 2025.12.

> **Source anomaly — 2025.12.1:** Listed under Current Documentation with a single updated
> Reference Guide PDF only. No full release package (ZIP, What's New, ReadMes) published for
> this patch — reference guide update only.

> **Source anomaly — 2024.10.2:** The version heading reads "Pro Tools 2024.10 / 2024.10.1 /
> 2024.10.2" but only ReadMes for 2024.10 and 2024.10.1 are listed. No 2024.10.2-specific
> ReadMe or documentation entry appears on the source page. ⚠️

> **Note — Plugin guide versions:** Cell instrument guides (PlayCell, GrooveCell, SynthCell)
> are dated 2023.7 — older than the current Pro Tools release. The Audio and MIDI Plugins Guide
> is dated 2024.10. These are not updated in step with Pro Tools core releases.

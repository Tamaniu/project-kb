# Avid | Stream IO 2025.2 — What’s New

> Version: 2025.2  
> Product: Avid | Stream IO  
> Scope: Explicitly stated new features and compatibility information only

---

## New Remote Console
- Updated Remote Console available as a standalone module.
- Refreshed user interface.
- Improved performance and responsiveness.

---

## SMPTE 2110 Full Support
- SMPTE 2110 ingest and playout are now fully supported.
- SMPTE 2110 ingest moves from technical preview to full certification.
- Enables production-ready IP-based video workflows.

---

## SMPTE 2022-7 Redundancy (Technical Preview)
- Technical preview support for SMPTE 2022-7 redundancy.
- Provides seamless failover between redundant streams.
- Intended for early testing and feedback prior to full release.

---

## Virtualized Deployment
- Avid Stream IO server is now supported in virtualized environments.
- Complements existing bare-metal installation support.
- Enables improved deployment flexibility, scalability, and resource optimisation.

---

## SuperMicro AS-115HS-TNR Certification
- Official certification added for the SuperMicro AS-115HS-TNR platform.
- Expands supported hardware options for Avid Stream IO installations.
- Refer to *Avid | Stream IO Hardware Specification & Guide v2025.2* for details.

---

## SRT Playout Support
- Support added for compressed IP-based playout using SRT (Secure Reliable Transport).
- Enables efficient and reliable content delivery over IP networks.

---

## Playlist Clip Override
- Existing playlist clips can be replaced at a later stage.
- Supported via Send to Playback (STP) with the **Overwrite** option enabled.
- Allows seamless playlist updates without manual reconfiguration.

---

# Media Composer 2019 — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base 2019 documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Media-Composer-2019-Documentation>

Page last updated: 11/02/2021.

> **Source anomaly — http:// URL:** Only the v2019.12.5 ReadMe uses `http://`. Normalised to
> `https://` in this index.

> **Note — Guide naming conventions:** This page uses different labels from 2020+ pages:
> "User's Guide" (not "Editing Guide") and "Effects Guide" (not "Effects and Color Correction
> Guide"). The underlying PDF filenames follow the same pattern however
> (`Media_Composer_Editing_Guide_2019.x.pdf`, `Media_Composer_Effects_and_CC_Guide_2019.x.pdf`).

> **Note — Distributed Processing:** Three separate DP Admin Guide PDFs are listed, one per
> major version point (v2019.12, v2019.9, v2019.6). The v2019.6 guide uses a generic unversioned
> filename (`MC_Distributed_Processing_Admin_Guide.pdf`). The v19.12 PDF is the same one linked
> from the 2020 documentation page.

> **Note — No VDI Nutanix guide:** The Nutanix Deployment Guide is absent from this page.
> It first appears on the 2020 documentation page (2021-edition PDF).

> **Note — FlexNet:** Only the legacy FlexNet Device Manager (v8.x) guide is listed.
> The Avid FlexNet License Server Manager Admin Guide is absent from this page.

> **Note — Enterprise Admin Guide:** Same unversioned PDF (`MC_Enterprise_Admin_Guide.pdf`) as
> the 2020 and 2021 pages.

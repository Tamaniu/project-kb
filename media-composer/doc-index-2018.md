# Media Composer 2018 — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base 2018 documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Media-Composer-2018-Documentation>

Page last updated: 23/02/2021.

> **Note — No http:// URLs:** All links on this page use `https://`. No normalisation required.
> This is the only documentation page in the 2018–2025 range with no http:// anomalies.

> **Source anomaly — Effects Guide filename:** The Effects and Color Correction Guide PDF uses
> the filename `Media_Composer_Effects_and_CC_Guide_18.x.pdf` — note the shortened year prefix
> `18.x` rather than the full `2018.x` used by all other years. The label on the source page
> correctly reads "v2018.x". ⚠️

> **Source anomaly — "v 2018.10" label:** The source page lists "Media Composer v 2018.10 What's
> New" with a space between "v" and "2018". Recorded as v2018.10 below.

> **Note — Absent guides:** No Enterprise Admin Guide, Distributed Processing Admin Guide,
> High-Resolution Workflows Guide, or VDI Nutanix guide is listed on this page. The DP Admin
> Guide first appears on the 2019 page (v2019.6 guide with generic filename).

> **Note — v2018.12.x patch depth:** v2018.12 reached patch level .15 — the highest patch
> count across all documentation pages in this index. Reflects the extended support life of
> the 2018 release line.

> **Note — FlexNet:** Only the legacy FlexNet Device Manager (v8.x) guide is listed.

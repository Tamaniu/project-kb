# New Features in MediaCentral | Data Receiver 2022.12

## New User Interface

### Streamlined UI
Changes can be made directly without navigating numerous dialogs. Visual aids guide the user, such as icons on fields requiring input, bold labels on tabs indicating configurable services, and a **Restore** button for easy rollback of changes.

### Service Description Tab
A new **Service Description** tab summarizes the content of the wire service.

### Printing and Exporting
Both **Print** and **Export** features allow easy backup of custom configurations.

### Importing
An easy-to-use **Import** feature simplifies re-creating wire services from saved configurations. It also supports importing previous versions of service configurations, automatically converting the old **SFC** processing format into the new **JavaScript** processing format.

### Cloning
The **Clone** feature enables quick duplication of configured wire services within the Data Receiver Server, allowing rapid setup of similar configurations.

### Monitoring
The Server and Administrator Application are designed so that the Administrator Application can run separately from the Server, enabling easier access to monitor or change wire services across different Data Receiver Servers.

### Encryption Support
Encryption and authentication support is updated to allow access to newer, more secure RSS feeds.

### Email Server Support
Email server support is enhanced to enable access to email servers through **IMAP** or **POP3** with encryption and authentication.

## New Process Language

### JavaScript
The process language is switched to **JavaScript**, a widely used and well-documented programming language.

Data Receiver includes a built-in converter to automatically convert legacy **SFC** processing files to the new JavaScript format during import.

# New Features in MediaCentral | Data Receiver 2023.3

## User Interface

### Reveal Icon
Data Receiver now provides a **Reveal** icon that allows users to view the password they have typed.

### Summary Tab
The **Summary** tab includes two enhancements:
- A new splitter between the **Services** and **Log** areas, allowing users to resize these sections.
- A new wire application log filter that enables filtering logs from **1 to 7 days**.

### Processing Tab
The **Processing** tab includes two improvements:
- A separate **Find** toolbar and **word-wrap** support for both Input and Output windows.
- Enhanced **input translation debugging**.

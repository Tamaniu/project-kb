# Editing

## Media Central Editor

Wolftech integrates with the Media Central Editor to enable creative workflows for editing content.

## Create Sequences from Wolftech Publishing Board

MediaCentral sequences are requested from a publishing board in Wolftech, allowing editors to work directly with story assets and publishing requirements.

## Admin: Connecting the Media Central Editor

To enable editing workflows, admins need to configure the integration between Wolftech and Media Central Editor in the system settings.

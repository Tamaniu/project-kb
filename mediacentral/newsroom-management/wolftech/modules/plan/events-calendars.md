# Events & Calendars

## Intro

Events are items happening in the future where you don't know if you will create a story out of it but are worthy of recording for future reference.

We have 2 types of events in Wolftech:
- Those generated internally (Internal Events)
- Those that we ingest automatically from 3rd party systems (External Calendar items)

## Internal vs External Events

### External Calendar Items
External calendar items are like external feeds, they cannot host bookings, conversations, or any further work. You can however create internal events, pitches, stories or episodes out of them, which will be listed in their card afterwards, as well as pin them.

### Internal Events
Internal events, on the contrary, have all the functionality from a story so all information can be planned and if you decide to create a story out of them all information will be transferred.

**Note:** The main difference between events and stories is that you cannot create publishing points out of events. All publishing comes from stories.

## Event Calendars Setup

Events, whether internal or external, are grouped in calendars. Internal calendars are defined by the admins. External events are grouped by the 3rd party system and defined in the workflow engine with the assistance of our professional services.

To create a new internal calendar admins can go to the admin → Calendar Types section where you can click on the 3-dot menu to create a new one.

Once created you can assign them a colour to clearly identify events belonging to this calendar.

## Quick Add and Categorisation

### Quick Add
To create an event the easiest way is to go to the Quick Add (orange plus icon button) and select Event. This can be done from the web or from the Wolftech Go app.

Here you will be able to name the event, give it a description, categorize it with a genre, select the calendar it belongs to and give it some dates and times.

# Plan View

The Plan module is at the heart of Wolftech's story planning. It is the town hall where teams get full visibility of all of the activity happening today, this week or this month across their part of the organization. It can track internal events, external calendars, pitches, stories, productions, tasks or publishing points filterable in every way you can think of.

In this guide we will help you to get to know its functionality deeply so you can make the most of it and are confident in setting them up if you are an admin.

**Note:** This guide is based only on the "new plan view". There are important differences with the old plan view which will not be covered in this guide. The old plan view will no longer be available from R14 onwards and we have chosen to focus the documentation on the new one as a result.

## Access

### Access to Plan Module

To see the plan view users will need to navigate to the PLAN module in the top menu of the application.

### Access to Shared Views

On login you will initially get access to your department's default collection of views. This means you are not required to configure the filters or load the collection yourself.

Provided a default collection is shared with you, as soon as you land on the Plan Module you will see the active View Name you are on in the top left (see "All Departments"). Next to it you will see a dropdown icon that will expand to display the other views you have access to as part of this shared collection of views.

If users have been given access to multiple shared collections of views we will first display the department's default collection but they will be able to load the other shared collections as needed. See Loading other shared collections section.

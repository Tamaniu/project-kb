# Wolftech Go

## Overview

Wolftech Go is the mobile application that extends Wolftech News capabilities to iOS and Android devices, enabling field journalists, reporters, producers, and crew to stay connected and productive while on location.

## Features

Wolftech Go provides:
- Real-time access to stories and productions
- Mobile task and booking management
- Location tracking and GPS integration
- Push notifications for assignments and updates
- Offline access to essential information
- Mobile-optimised user interface

## Installation

The Wolftech Go mobile app is available for:
- iOS devices via the Apple App Store
- Android devices via the Google Play Store

Users can access the app download link from their My Page profile menu in the main Wolftech application.

## Configuration

To connect the mobile app to your Wolftech database:
1. Open the Wolftech Go app
2. Scan the QR code displayed in the My Page settings section of the main application
3. Alternatively, enter your server URL and authenticate via the identity server

**ProTip:** If you need to reset the QR Code and reconnect to the database, go to the hamburger menu on the left and choose About. Then scroll down to the reset option and scan the QR code again.

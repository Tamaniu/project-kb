# Long Form Product Overview

## Intro

The Long Form Module is a Wolftech Add-On that enables a number of functionalities to help teams manage long form productions within their existing Wolftech News story planning product. This guide aims to introduce the Long Form Module in detail. It is aimed at admins but it will cover both administrative duties as well as general functionality so admins and super users feel confident training teams internally.

## Access – User Types

The long form module is an add-on that is activated at the account level. Everyone who gets access to Wolftech News as part of the account will therefore be able to leverage the long form module. The same SSO and authorisation protocol works. No changes required.

For further details see the user management guide.

The application can be accessed either through your browser at your own company defined url or via the mobile app. Typically the web interface is used by managers, coordinators, planners, and editors and the mobile app becomes the primary interface for journalists, reporters, producers in the field and related crew.

## Access – Web – Home Dashboard

As you log in to your web application you will land in the Home Dashboard, where you can see at a glance all of the most relevant information for your day to day work.

It contains:
- A left menu (which you can explore in the user setup guide)
- A top central menu to navigate through the main planning tools in the system
- A top right search and communication bar
- A top right profile (personal menu)
- A bottom area where you may receive editorial alerts
- And a central dashboard with number of panes with different information

**ProTip:** Some users may ask if you can reorganise the panes in the home dashboard. This is not possible on the web but it is possible on the mobile app.

For a full guide on the different workflow elements within the Long Form Module, please refer to the subsequent sections of this document.

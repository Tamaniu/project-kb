MediaCentral | Panel for 3rd Party Creative Tools Version 2024.10.1 ReadMe Avid recommends that you read all the information in this ReadMe file thoroughly before installing or using any new software release.
This document describes the integration between MediaCentral and Adobe Premiere Pro, including support for Adobe Photoshop and After Effects.
Follow the links on the following Avid Knowledge Base page to obtain more information on your specific version of MediaCentral Cloud UX:
http://avid.force.com/pkb/articles/en_US/user_guide/MediaCentral-CloudUX-Documentation Revision History Date Revised Changes Made March 31, 2025 First publication Contents About MediaCentral | Panel for 3rd Party Creative Tools About MediaCentral | Panel for 3rd Party Creative Tools Adobe users can now access the power of the Avid MediaCentral Platform from within Adobe Premiere Pro using a dedicated panel (the Panel for 3rd Party Creative Tools). The Panel also includes support for Adobe Photoshop and Adobe After Effects.
For MediaCentral Cloud UX users, the panel enables a roundtrip workflow with Avid’s asset management systems by connecting an Adobe Premiere Pro editor to Avid MediaCentral | Production Management and MediaCentral | Asset Management. Adobe Photoshop and After Effects users can store their photos, clips, graphical elements and other content for access anytime on the open MediaCentral platform.
Editors working in Adobe Premiere Pro can access production assets and associated metadata across Production Management and Asset Management systems using MediaCentral Cloud UX. They can import and edit MediaCentral Cloud UX basic and advanced sequences in a variety of Adobe Premiere Pro supported formats directly from Avid shared-storage workspaces. Using the MediaCentral | Panel for 3rd Party Creative Tools, Adobe Premiere Pro editors can also export the active sequence from Adobe Premiere Pro to MediaCentral Production Management or MediaCentral Asset Management in the format and resolution of choice.
And graphic designers can store their assets and projects and catalog them, enabling teams to easily access, edit on-the-fly and send back content into Avid production workflows. This allows teams to instantly locate individual design elements in archived content, eliminating productivity drains and having to save files on a separate storage platform.
This guide references elements of the MediaCentral Cloud UX user interface, such as the Fast Bar, the Asset Editor, the Browse and Search apps, and more. Users are expected to have a good working knowledge of the MediaCentral Cloud UX software.
If you need more information about these or other areas of MediaCentral Cloud UX, see the Avid MediaCentral | Cloud UX User Guide.
New Features in v2024.10.1 This section covers the new features for the MediaCentral | Panel for 3rd Party Creative Tools in the 2024.10.1 release.
Cancel Import Prompt You now have the option to Cancel or Wait when importing a sequence into Adobe Premiere Pro if the operation takes a long time to complete. A dialog box will appear asking if you want to Cancel or Wait to continue the process after a significant amount of time has passed while waiting for the import to finish.
Fixed in Version 2024.10.1 The following issues have been resolved in MediaCentral Panel for 3rd Party Creative Tools.
Bug Number: AL-3498. Attempting to import a complex sequence into Adobe Premiere Pro might fail with a 'Could not get AAF path, reason: Error: avid.pam.extender service returned code 200, message OK' error.

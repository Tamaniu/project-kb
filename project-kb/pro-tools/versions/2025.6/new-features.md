3 Transcription Pro Tools Studio and Ultimate provide tools to transcribe speech from audio to text. This is especially useful for editing dialog in post-production, making radio edits, and even song writing.
When transcribing text from audio, the Pro Tools Speech to Text Engine analyzes the entire audio file.
Transcription data is stored in the header for WAV files or separately in the session folder for other file types. Note that since transcription requires analysis of entire audio files, it may take some time for the transcription to complete. You can monitor the progress of the transcription in the Task Manager. Also, note transcription occurs in the background so that you can continue to work in Pro Tools on other tasks.
g Pro Tools Speech to Text Engine is installed separately from Pro Tools. If you attempt to transcribe text from audio, but the library is not installed, you are prompted to install it.
Transcription is only available with Pro Tools Studio and Pro Tools Ultimate.

- Search, preview, access, and use content that is not on the timeline by text.
g An internet connection is not required to use Transcription.

8 New MIDI Plugins TATAT from K-Devices Pro Tools includes TATAT, an (unstable) streams generator MIDI plugin from K-Devices. TATAT is a MIDI FX plugin designed to generate, process, record, and export a stream of notes.
Use TATAT to:
Galways-changing sequences
- Quickly sketch and store melodic/harmonic ideas
- Add unexpected events to existing patterns and clips
- How TATAT Works
TATAT generates infinite and always changing sequences of notes. You cannot say exactly what TATAT will do, but you can suggest a path to it. By setting up its parameters, you can create what we call a "mood" for TATAT. A mood is a configuration of notes, intervals, time fragmentation, velocity fluctuation, and other factors.
Once you set a mood that you are happy with, you'll listen to an infinite stream of events, always changing, all staying in the mood.
At this point, according to your needs, you can:
- Let TATAT play its stream, and make some intervention to control it: to add silence (Less), to obtain
smooth accelerando and ritardando (Ratio), to control how TATAT generates intervals. You can enable Key In function to change notes on the fly (with a keyboard, or just feeding TATAT with a MIDI clip).
Add some stability by controlling/automating the Liv/Mem slider, moving between the generated
- stream and frozen patterns.
- Use TATAT to send its notes to enrich (by enabling Thru) existing/incoming MIDI data/parts.
- Export (desktop version only) a nice mood as a series of MIDI files/clip: just click on the export icon
and drag it in your session or where you prefer. For each time you’ll drag it, TATAT will create a new variation of the mood! This is a great feature to create a set of coherent ideas, lines, or riffs as a starting point for a new project.

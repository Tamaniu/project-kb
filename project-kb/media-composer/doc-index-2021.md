# Media Composer 2021 — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base 2021 documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Media-Composer-2021-Documentation>

Page last updated: 17/04/2024.

> **Source anomaly — http:// URLs:** The majority of links on this page use `http://` rather than
> `https://` — specifically all v2021.12.x ReadMes, v2021.9, v2021.6, v2021.12 What's New, the
> v2021.x Editing Guide, and both Distributed Processing Admin Guides. URLs have been normalised
> to `https://` in this index, as `resources.avid.com` redirects HTTP to HTTPS.

> **Source anomaly — Install Guide:** The Install Guide on this page links to
> `Install_Guide_Editor_v2020.x.pdf` — a v2020.x-era file, not a 2021-specific guide. This
> appears to be an oversight in the source page. ⚠️

> **Note — Enterprise Admin Guide:** This page lists an unversioned guide
> (`MC_Enterprise_Admin_Guide.pdf`) updated June 2021. Later pages (2022+) link to year-versioned
> files (e.g. `MC_Enterprise_Admin_Guide_2022.x.pdf`). These are distinct PDFs.

> **Note — High-Resolution Workflows Guide:** This guide (FrameFlex and ACES workflows) appears
> on this page only; it is not listed on the 2022–2025 documentation pages.

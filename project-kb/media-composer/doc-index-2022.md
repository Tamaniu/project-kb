# Media Composer 2022 — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base 2022 documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Media-Composer-2022-Documentation>

Page last updated: 01/08/2024.

> **Source anomaly noted:** Several URLs on this page use `http://` rather than `https://` — specifically
> the v2022.4 readmes, v2022.7 readmes, Install Guide, v2022.x Editing Guide, FX Guide, Enterprise Admin
> Guide, and all three Distributed Processing entries. URLs have been normalised to `https://` in this index,
> as `resources.avid.com` redirects HTTP to HTTPS. The source page links are preserved accurately otherwise.

> **Note:** FlexNet, VDI, and VMware supplemental entries share the same PDF URLs as the 2023–2025
> documentation pages. The Enterprise Admin Guide URL is distinct (v2022.x).

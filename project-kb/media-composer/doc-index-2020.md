# Media Composer 2020 — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base 2020 documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Media-Composer-2020-Documentation>

Page last updated: 07/07/2025.

> **Source anomaly — http:// URLs:** v2020.12.2 through v2020.12.8 ReadMes use `http://`
> rather than `https://`. URLs have been normalised to `https://` in this index, as
> `resources.avid.com` redirects HTTP to HTTPS.

> **Source anomaly — "Media Composer v2020 ReadMe" label:** The entry labelled
> "Media Composer v2020 ReadMe" links to `README_Avid_Editor_v20.4.pdf`. This appears to be
> the v2020.4 initial release ReadMe relabelled as "v2020"; it is grouped under v2020.4 below.
> ⚠️

> **Source anomaly — Enterprise Admin Guide date:** The source page lists the Enterprise Admin
> Guide as updated "January 19, /2021" — the forward slash appears to be a typo in the source.
> Recorded here as 19/01/2021.

> **Source gap — v2020.12.4:** No ReadMe for v2020.12.4 is listed on the source page
> (sequence runs 12.3 → 12.5). This gap is in the source, not this index.

> **Note — Distributed Processing:** This page links to
> `MC_Distributed_Processing_Admin_Guide_v19.12.pdf` — a v19.12-era guide. No 2020-specific
> DP Admin Guide PDF is listed.

> **Note — VDI Guide:** This page links to `MC_VDI_NutanixGuide_2021_02.pdf`, which is distinct
> from the 2022-edition PDF (`MC_VDI_NutanixGuide_2022.pdf`) listed on the 2021–2025 pages.

> **Note — FlexNet:** Only the legacy FlexNet Device Manager (v8.x) guide is listed. The newer
> Avid FlexNet License Server Manager Admin Guide (`Avid_FlexNet_License_Server_Administration_Guide.pdf`)
> is absent from this page.

> **Note — Enterprise Admin Guide:** Same unversioned PDF (`MC_Enterprise_Admin_Guide.pdf`) as
> the 2021 page. Distinct from the year-versioned files introduced from 2022 onwards.

> **Note — High-Resolution Workflows Guide:** Present on this page, as on 2021. Not listed on
> the 2022–2025 documentation pages.

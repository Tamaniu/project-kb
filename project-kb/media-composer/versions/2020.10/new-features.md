New in Media Composer v2020.10 What’s New for Avid® Media Composer® v2020.10 New in Media Composer v2020.10 The following are new for Media Composer v2020.10 • macOS Support (cid:129) ProRes RAW macOS Support This release includes support for macOS v10.15.7.
ProRes RAW Media Composer supports linking and playback of ProRes RAW media using the UME plug-in.
Exporting ProRes RAW is not supported.
IMPORTANT:
(cid:129) For macOS v10.14 and earlier, make sure the “Pro Video Formats” package is installed (if not already present). Download available here.
(cid:129) For Windows 10 64-bit systems, make sure you have the Apple SDK installed on your system.
Download available here.

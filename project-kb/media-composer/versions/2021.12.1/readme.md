Avid Editing Application ® ReadMe for Media Composer v2021.12.1 Revision History Date Description April 8, 2022 Removed support for macOS v12.3. Added fixes to Fixed in Media Composer v2021.12.1.
April 7, 2022 Added fix in Fixed in Media Composer v2021.12.1 April 7, 2022 Release for Media Composer v2021.12.1 Important Update This release of Media Composer does not support macOS v12.3.
Fixed in Media Composer v2021.12.1 The following have been fixed:
a Bug Number: MCCET-4330. Background transcoding of ProRes RAW files to another format failed to complete.
a Bug Number: MCCET-4558. Media Composer Enterprise was not able to launch when using Avid NEXIS client software.
a Bug Number: MCCET-4570. In some instances, when working in a standalone environment (local drives), you could not successfully perform a Send To Playback.
a Bug Number: MCCET-4563. When logging in to Production Management or MediaCentral | Cloud UX from the Media Composer project window, the password remained in the password text box after you logged out.
Fixed in Media Composer v2021.12.1 a Bug Number: MCCET-4465. In some cases Media Composer options would not be seen in Media Composer when Media Composer used Cloud Licensing.
a Bug Number: MCCET-4535. Using keyboard shortcut to open multiple bins resulted in floating panels instead of tabbed bins.
a Bug Number: MCCET-4527. Exporting BXF files caused a crash.
a Bug Number: MCCET-4442. JKL playback did not work properly when switching workspaces with Calculator tool open.
a Bug Number: MCCET-4577. Saving a new bin view made bin columns wider.
a Bug Number: MCCET-4485. Insert Export was not working properly.
a Bug Number: MCCET-4265. A sequence containing grouped XDCAM 50 clips with FrameFlex did not retain FrameFlex after performing a mixdown.
a Bug Number: MCCET-4490. Sorting by Date in the Find window did not sort correctly when Regional Format was set to English (United Kingdom).
a Bug Number: MCCET-4361. You could not capture from the front XLR input of the Avid Artist DNxIO.
a Bug Number: MCCET-4378. (NDI) In some instances, the Media Composer monitor was either behind or ahead of the NDI stream.
a Bug Number: MCCET-4242. When working with some AudioSuite plug-ins, rendering stereo audio was slow in a Production Management/Avid NEXIS environment.
a Bug Number: MCCET-4360. In some instances, exporting to a P2 device resulted in pixelated and inverted images.
a Bug Number: MCCET-4399. If you performed an Export as Graphic and NDI was active, you might have received an “AMEResourceManager ReconfigHW_LegendBoB_Capture” error.
a Bug Number: MCCET-3854. During export some AAF Sound settings did not save.
a Bug Number: MCCET-4481. (Production Management) Clips with the Restrictions did not appear with the red triangle in the Media Composer bin.
Fixed in Media Composer v2021.12.1 a Bug Number: . MCCET-4497 (macOS) Performing a Select Media Relatives on a sequence crashed the application.
a Bug Number: MCCET-4509. In some instances, the image appeared garbled after loading a sequence in a Color Correction workspace.
a Bug Number: MCCET-4449. In some instances, when working with MultiCam sequences, Media Composer was sluggish.
a Bug Number: MCCET-4402. With Gang enabled, selecting all Timeline Clip Notes resulted in a hang.
a Bug Number: MCCET-4395. (ScriptSync) If you added color to a range of script indicators, when you closed and then reopened the script, the indicators were filled even if they were not previously filled.
a Bug Number: MCCET-4499. (macOS) Using the Enter key to Adjust Pan / Volumes IN to OUT did not work.
a Bug Number: MCCET-4513. EWC media in a Production Management environment could cause sluggishness in the editor this has been resolved.
a Bug Number: MCCET-4498. Lassoing in Bin Frame View using Shift or Ctrl did not work properly.
a Bug Number: MCCET-4056. Applying a Matte key effect to a ProRes 4444 file did not preserve the fill layer.
a Bug Number: MCCET-4437. Selecting “Set Audio Suite Plugin In/Out” from the AudioSuite Plugin fast menu does not work on rendered effect after performing an Undo.
a Bug Number: MCCET-4519. In some circumstances you might have received an error when using Color Correction with dupe detection enabled.
a Bug Number: MCCET-4382. In some instances, Dynamic Relink settings unexpectedly reverted to Most Recent.
a Bug Number: MCCET-4562. In some instances, LTC signals did not parse correctly in Read Audio Timecode.
a Bug Number: MCCET-4332. Transcoding linked 50fps h264 clips to 50i resulted in the wrong duration.
Fixed in Media Composer v2021.12.1 a Bug Number: MCCET-4371. (macOS) When working with a vector based object, Shift- clicking a direction handle did not create an asymmetrical handle.
a Bug Number: MCCET-4479. When moving bins in a project, it can take too long for the bin to respond and in some instances the beachball appears when moving the bin.
a Bug Number: MCCET-4443. You might have received a “m_qDrag deleted with deleteLater” error after dragging a button from the Command Palette to another button under the Composer Window.
a Bug Number: MCCET-4501. If a floating Timeline window is made smaller, scrolling with the mouse wheel does not work properly.
a Bug Number: MCCET-4450. Media Composer | Enterprise might have launched as Media Composer standard.
a Bug Number: MCCET-4436. If Dynamic Relink was enabled, an imported audio file might have appeared offline.

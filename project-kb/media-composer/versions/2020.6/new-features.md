New in Media Composer v2020.6 What’s New for Avid® Media Composer® v2020.6 New in Media Composer v2020.6 The following are new for Media Composer v2020.6:
• Refresh Sequences Update (cid:129) Displaying Titler+ Titles in Frame Views (cid:129) Variable Frame Rate Media Support (cid:129) Timeline Segment Selection Info in The Inspector (cid:129) Dock or Tab the Settings Window (cid:129) Improved Lassoing in Frame View New in Media Composer v2020.6 Refresh Sequences Update The option “All FrameFlex including creative framing” has been added to the Refresh Sequences options.
In some cases, such as if you are editing a horizontal sequence to use as the base for a vertical sequence, you might want to force a refresh including the FrameFlex changes made in the Timeline.
Displaying Titler+ Titles in Frame Views When your bin is in Script View or Frame View, Titler+ title presets display a still of the first frame in the bin. The title is displayed over black.
> **Note:** If you have a fade-in in the preset, no title is present to show. You will only see black.
Variable Frame Rate Media Support Media Composer now supports variable frame rate media wrapped in .MOV and .MP4 without the need for QuickTime.
Timeline Segment Selection Info in The Inspector When you select one or more segments in the Timeline, the Inspector tool will now provide specific metadata about those segments. When you select multiple segments in the Timeline, the Inspector Tool will display the common metadata across all in the selection. If there are different values in the same field, “Multiple Selected” displays in the Name field and “Multiple Values” displays in all others.
> **Note:** The information displayed for Timeline selection is fixed. You cannot choose columns like you can for
bin information.
New in Media Composer v2020.6 Dock or Tab the Settings Window The Settings Window can be docked or tabbed. To tab the window, press the Alt key while dragging the vertical title bar of the window you want to tab to another window. To dock the window, Click in the vertical title bar of the Settings window and drag to the location where you want to dock the window. Locations where you can dock the window will appear as green drop zones.
Example of tabbed Settings window For details on tabbing and docking windows, see “The User Interface” in the Help.
Improved Lassoing in Frame View With this release, when you are in Frame View in a bin, the bin will scroll during lassoing to allow you to easily select multiple clips.

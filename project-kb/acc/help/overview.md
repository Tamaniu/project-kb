# Avid Content Core — Overview

**Source:** docs.avid.com ACC Help — Welcome  
**Extracted:** 01/05/2026 | Docs version: 0.0.3

---

## What is Avid Content Core?

Avid Content Core (ACC) is a SaaS-based asset discovery and management system combining Avid's media industry experience with AI technology and cloud scalability. It empowers individuals within an organisation to create and deliver stories with greater speed, clarity, and impact.

ACC is a **Content Data Platform (CDP)** scalable from small businesses to large globally dispersed enterprises — for securely viewing, managing, editing, and monetising assets.

---

## Key Features

**Easy to Use, Massive Reach** — Single point of access to assets from any connected site and system. Breaks down siloed sites so creatives can leverage assets created from anywhere at any time.

**Online Playback** — The ACC workflow automatically creates cloud-based proxy media for new assets, removing the need to wait for high-res media transfer before working.

**AI-Driven Efficiency** — ACC automatically provides collections of assets based on usage history and recent activity of individuals in the team.

**Natural Asset Discovery** — Natural language, federated search finds assets across the entire organisation without requiring text-based metadata knowledge.

**Real-Time Collaboration** — Assets can be combined in a sequence editable by multiple simultaneous users.

**Editing and Multi-Platform Publishing** — Built-in editing and publishing tools produce a single finished product distributable across platforms and screen aspect ratios.

---

## What is SaaS and Why Does it Matter?

*SaaS* (Software as a Service) is a centrally managed, truly cloud-native software solution. Key advantages for media organisations:

- **Reduced local infrastructure** — less requirement for power, cooling, and dedicated server rooms.
- **Low administration overhead** — no client software to install or maintain on individual workstations; no on-prem server management.
- **Continuous updates** — Avid deploys new features automatically without dedicated maintenance windows or lengthy upgrade planning.
- **Browser-based UI** — accessible through a standard web browser (Google Chrome) from any workstation with internet access.

---

## SaaS vs On-Prem — Practical Implications

| Aspect | Traditional On-Prem | ACC SaaS |
|---|---|---|
| Client software | Installed per workstation | Browser only (Chrome) |
| Server maintenance | Engineering team required | Managed by Avid |
| Feature releases | Planned maintenance windows | Continuous, automatic |
| Infrastructure | Local servers, power, cooling | Cloud (AWS + GCP) |
| Access | VPN or local network | Public internet (port 443), no VPN needed |

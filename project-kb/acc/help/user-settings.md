## Accessing User Settings

1. Click the **User Menu** (upper-right corner) and select **Settings**.
2. After making changes, click **Save**. Click **Revert** to return to the previous state.
3. To close: click outside the dialog, click the **X** (Close) button, or press **Esc**.

---

## Data Collection

Opt in or out of Avid's usage data collection, which is used to improve the ACC experience (particularly Landing Page asset recommendations).

- **Enabled (blue toggle):** Avid collects usage data; Landing Page recommendations are personalised.
- **Disabled (black toggle):** No data collected; only group-level trends shown on the Landing Page.

This preference can be set at first sign-in and updated at any time.

For information on data collected: https://kb.avid.com/pkb/articles/en_US/Knowledge/Usage-Data-Sharing-In-Avid-Content-Core

**To configure:**
1. Open User Settings. Select **Data Collection** from the navigation panel.
2. Toggle to enable or disable. Click **Save**.

## General Settings

Applies to dates and times displayed throughout the ACC UI.

| Setting | Description |
|---|---|
| Time Zone | Controls how dates and formatted data are displayed. Default: "Use System Timezone" (from workstation OS). |
| Date Format | Options: DMY `DD.MM.YYYY HH:mm:ss Z` (Alternative) · DMY-slash `DD/MM/YYYY HH:mm:ss Z` (European) · YMD `YYYY-MM-DD HH:mm:ss Z` (International) · MDY `MM/DD/YYYY hh:mm:ss A Z` (US). `Z` = UTC offset. |

**To configure:**
1. Open User Settings. Select **General** from the navigation panel.
2. Select a Time Zone and Date Format. Click **Save**.

## Notification Settings

Configures which types of messages are delivered through the Notifications app.

Configurable notification categories:

- **Review and Approval**: notifications for requested, rejected, approved, or cancelled R&A tasks.
- **User Tasks**: notifications for created, in-progress, and completed user tasks of access type "Team" *(reserved for future use)*.

Settings are divided into: All notifications / App / Feature categories. Each app can be expanded (click the chevron) to show feature-level settings.

**To configure:**
1. Open User Settings. Select **Notifications** from the navigation panel.
2. Configure notifications:
   - Toggle an **individual feature** to enable/disable (blue = enabled).
   - Toggle the **app header** to enable/disable all notifications for that app.
   - Toggle **All Notifications** to enable/disable all notifications across all apps and features.
3. Click **Save**.

## Search Settings

Applies to the Search app and the Search bar at the top of the ACC UI.

| Setting | Description |
|---|---|
| Query Mode | **Compact** (default): search terms displayed as "pills". **Form**: each term displayed in full detail across two columns. |

Default Query Mode can be set here and overridden per session in the Search app header.

**To configure:**
1. Open User Settings. Select **Search** from the navigation panel.
2. Select **Compact** or **Form**. Click **Save**.

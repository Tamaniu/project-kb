# Compatibility Support Matrix: Media Composer | Cloud Remote

> ⚠️ **End of Life — Historical Reference Only**
>
> Media Composer | Cloud Remote is no longer available for sale as of 30 September 2022. MediaCentral v2022.3 is the last version to include support for Cloud Remote. Cloud Remote is no longer under active development.
>
> - **End of Development:** 15 September 2022
> - **Final Sale Date:** 30 September 2022
> - **End of Support Date:** 31 July 2023
> - **Final MediaCentral version with support:** v2022.3

Last revised: 27/10/2022

## MC | Cloud Remote Playback v2018.6

| Component | Supported Versions |
|---|---|
| Media Composer [a] | v2018.7–v2018.12.x [b]; v2019.12.x–v2022.x [c] |
| MediaCentral Cloud UX | v2018.6–v2022.3 |
| MediaCentral Platform Services | v2.10.4.3 to v2.10.4.x; v2.10.5.2 and later |
| MC Production Management [d] / Avid Interplay Production [e] | See Avid Editor Compatibility with Interplay Releases matrix |

**Footnotes:**

a. If you plan to use the MC Panel for Avid Media Composer, you must also consider version compatibility between MC and MC Cloud UX, which might be greater than the versions listed here. See the MediaCentral Compatibility Matrix.

b. Windows 10 or macOS required (macOS 10.15 and higher not supported). If running Production Management v2019.6, you must install the Production Management v2018.11.0 (Win) or v2018.11.3 (macOS) client components.

c. Windows 10 or macOS required (macOS 10.15 and higher not supported). If running Production Management v2019.6, you must install the Production Management v2018.11.0 (Win). You must also install and launch Interplay Access 2019.6.2 at least once somewhere in your network.

d. Do not install Production Management v2018.11.x client patches on your MC Cloud Remote editor as these patches are incompatible with the Cloud Remote workflow.

e. Requires Interplay Production 3.8.x or later to enable all MC Cloud Remote Playback v2018.6 features.

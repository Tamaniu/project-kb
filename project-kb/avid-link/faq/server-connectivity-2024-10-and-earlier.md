# Avid Link 2024.10 and Earlier — Server Connectivity End-of-Life FAQ

Source: Avid Knowledge Base — "Avid Link 2024.10 and earlier will not connect to the server | FAQ"  
Source last updated: 28/04/2026  
KB doc added: 29/04/2026

> **Action required before 12/05/2026.** Avid Link versions 2024.10 and earlier will lose the ability
> to connect to Avid's servers from 12 May 2026. Update to version 2025.5 or later (2026.4 or later
> recommended) before that date to avoid disruption. The update is free.

---

## Summary

Beginning **12/05/2026**, Avid Link versions **2024.10 and earlier** will no longer be able to connect
to Avid's servers. This is part of ongoing infrastructure maintenance to improve security and
reliability. License activation is **not** affected — installed Avid products will continue to run.

---

## Applies To

- Avid Link on **Windows 11**
- Avid Link on **macOS 14 and later**
- Users of **Sibelius**, **Pro Tools**, and **Media Composer** who sign in through Avid Link

---

## What Changes on 12/05/2026?

Users running Avid Link 2024.10 or earlier after 12/05/2026 will **lose** the ability to:

- Sign in to their Avid account from Avid Link
- Access Pro Tools cloud projects (now available directly in Pro Tools)
- Purchase products or subscriptions from the Marketplace
- Receive product news, release announcements, and product update notifications

**Not affected:** Licence activation. Installed Avid products (Sibelius, Pro Tools, Media Composer)
will continue to run. New product updates will not be delivered through Avid Link until the user
upgrades.

---

## Supported Versions

| Status | Version |
|---|---|
| Minimum supported | Avid Link 2025.5 |
| Recommended | Avid Link 2026.4 or later |

**Supported operating systems:** Windows 11; macOS 14 or later.

## How to Check Your Current Avid Link Version

1. Click the **Avid Link icon** in the menu bar (top of screen on macOS) or in the system tray
   (bottom right on Windows).
2. Choose **Show**.
3. Open the **Products** page. Your Avid Link version is displayed on the right.

If the version shown is **2024.10 or earlier**, upgrade before **12/05/2026**.

---

## How to Update Avid Link

You can download the latest version from either of the following:

- **Inside Avid Link** — from the Products page
- **Avid website** — https://www.avid.com/products/avid-link

During installation, Avid Link will restart to complete the update. No uninstall of the current
version is required — installing the latest version updates the existing installation in place.

---

## Enterprise and Offline Users

Enterprise customers or users on standalone or alternative licensing systems can continue running
older versions of Avid Link with **local network connectivity only**. No action is required for
offline workflows.

---

## FAQ

**Will my Avid products stop working if I don't update?**  
No. Licence activation is unaffected. Sibelius, Pro Tools, and Media Composer will continue to run.
You will lose access to sign-in, the Marketplace, Pro Tools cloud projects, and product update
notifications through Avid Link until you upgrade.

**Is the update free?**  
Yes. Updating Avid Link is free for all users, regardless of product or licence type.

**Do I need to uninstall my current version first?**  
No. Installing the latest version of Avid Link will update the existing installation.

**What happens if I update after 12/05/2026?**  
You can update at any time. Once you are on version 2025.5 or later, full connectivity to Avid's
servers will be restored.

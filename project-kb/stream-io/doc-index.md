# Avid | Stream IO — Documentation Index

Official PDF documentation index sourced from the Avid Knowledge Base Stream IO documentation page:
<https://kb.avid.com/pkb/articles/en_US/Knowledge/Avid-Stream-IO-Documentation>

Page last updated: 10/04/2026.

> **Note — ReadMe scope:** Per the source page, ReadMes are cumulative and updated with each
> patch. The latest ReadMe for a given major release includes all fixes and updates for that
> entire release including previous patches. ReadMes also incorporate What's New content —
> there are no separate What's New PDFs for Stream IO (contrast with Media Composer).

> **Source anomaly — v2024.10 FastServe Gen 2 guide filename:** The "Preparing FastServe Gen 2
> Servers" guide for v2024.10 uses a generic unversioned filename
> (`FS_Gen2_Conversion_Guide.pdf`), unlike all other versions which include a version suffix.
> ⚠️

> **Source anomaly — Hardware Guide filename prefix:** Versions v2023.8, v2023.11, and v2024.2
> use the prefix `Avid_Stream_IO_HW_Guide_` for the Hardware Guide PDF. From v2024.6 onwards
> the prefix changes to `Avid_Stream_IO_HG_`. These are distinct filenames.

> **Source anomaly — v2024.6 guide versioning:** The User's Guide and Hardware Guide for
> v2024.6 are versioned to patch level in their filenames (`v2024.6.1`) rather than major.minor
> only as with all other versions.

> **Note — Filename version format:** Newer versions (v2025.x, v2026.x) use underscores in
> the version component of PDF filenames (e.g. `2025_10`, `2026_3`). Older versions (v2023.x,
> v2024.x) use dotted version strings (e.g. `v2023.11`, `v2024.10`).

> **Note — FastServe Gen 2 conversion guide:** Present from v2024.10 onwards. Absent from
> v2024.6 and earlier. Covers preparation of FastServe Gen 2 server hardware for Stream IO.

# Avid Debuts Intelligent, Production-Proven Media Workflows Running on AWS at NAB Show 2026

**Source:** Avid Press Room
**Date:** 07/04/2026
**Location:** Burlington, MA

> Avid demonstrating how its media production solutions running on AWS are accelerating the
> industry's transition toward intelligent, cloud-native production.

---

> **Note on Content Core GA timing:** This press release (07/04/2026) describes the NAB Show

## Key Announcements

### NAB Show 2026 Debut — Avid Content Core
Avid Content Core makes its NAB Show debut, positioned as a cloud-native foundation for connected media workflows built on AWS. Target customers: news organisations, broadcasters, studios, and streaming services.

### Renewed Focus on Avid NEXIS on AWS
A dedicated push to bring Avid NEXIS, via its Virtual File System (VFS), into software-defined cloud and hybrid deployment — extending beyond fixed on-premises hardware.

### Amazon MGM Studios — Customer Reference
Amazon MGM Studios has deployed Media Composer and Avid NEXIS on AWS, enabling production teams to launch editorial environments globally in minutes and support distributed creative teams across multiple locations.

## Avid NEXIS on AWS — Capabilities

The NEXIS Virtual File System (VFS) enables expansion from on-premises into cloud and hybrid environments. Three deployment modes described:

| Mode | Description |
|---|---|
| Full cloud | NEXIS Cloud + Media Composer on Amazon EC2 and Amazon S3 — full cloud-enabled post-production |
| Hybrid | NEXIS Cloud unified with on-premises environments via **Avid NEXIS \| Remote** |
| On-premises | Traditional NEXIS deployment unchanged; cloud capability is additive |

**Key benefits cited:**
- Elastic performance and global deployment
- Secure remote collaboration
- Preserves trusted editing workflows including Media Composer Shared Projects and real-time collaboration
- AWS provides scalability, security, and compliance
- Customers can modernise at their own pace without disrupting creative processes

## Avid Content Core — "Intelligence Layer for Media"

Content Core is described in this release as Avid's evolution from cloud-enabled workflows to "a more intelligent and connected production environment." Key framing:

- **"The first true content data platform"** — unifies production into a single intelligent layer
- Brings together content, metadata, and workflow orchestration across creative, operational, and business processes

**Capabilities enabled:**

- **Interoperability:** AI-first architecture connecting Avid and third-party tools
- **Media understanding:** Persistent content identity and enriched metadata
- **Workflow efficiency:** Reduced friction across planning, production, and post-production
- **Modernisation pace:** Organisations connect siloed systems at their own pace

**NAB Show 2026 demonstrations:**
- Cloud and hybrid production at scale (NEXIS + Content Core)
- Modern newsroom and post-production workflows (Content Core)
- Unified foundation for AI and automation (Content Core)
